# 满语的叙利亚文字书写方案 ܡܢܙܘ ܣܘܪܝܐ ܗܪܓܢ

现代满文字的来源可以追溯到叙利亚文字。经过粟特、回鹘、蒙古文字演化以及新满文圈点改革，满文可以相对准确地记录满语，但是也积累了许多不合理之处。

在这里我们探究一种叙利亚文字对满语的转写方式，试图更加简洁地记录满语。

如果您对使用叙利亚文字的近亲文字阿拉米/希伯来文字书写满语感兴趣，请移步[מנזו אשׁורי/יברי הרגן](https://github.com/Gs-Linguashop/ManchuAramaicHebrew.git)。

叙利亚一系的文字是从右向左书写的。在传入西域时效仿汉文字改为从上往下书写。事实上，就是把原来的文字方向逆时针旋转了90度。这也是满文在换行时是向右换行，而不是类似汉语向左换行的原因。在现代信息化技术支持不足以纵向显示的时候，满文被迫从左向右显示，相比其根源的叙利亚文字整整旋转了180度，所以横向显示的满文才看起来好像上下颠倒了。

阿拉米一系的文字一共有22个辅音字母，其中ܐ（aleph）、ܝ（yodh）和ܘ（waw）也被用来记录a、i、u三个元音。此外，元音常常在辅音字母上下用可省略的标记来记录。基于这个系统，我们可以设计出满语的拼写方案。

## 满语叙利亚字：完整拼写 ܓܘܼܠܗܘܼܢ ܗܹܐܪܓܹܐܢ

满语有20个传统辅音音素、6个传统元音（a/e/i/o/u/ū）、复元音，以及来自汉语的辅音r/z/c、元音ü和音节si/shi/zhi/chi。

完整拼写方案完整地标注出满语的全部音素，一般用于拼写专有名词、外来词和需要详细注音的单词。

## 元音

叙利亚文字中，元音的拼写由辅音ܐ/ܘ/ܝ和元音上下标结合完成。与辅音拼写时，拼写方式略有不同。此外，元音a/e/o在不使用完整拼写时有时可以省略ܐ/ܘ拼写。

| 元音 | 与辅音拼写 | 省略拼写 | 穆麟德转写 |
| --- | --- | --- | --- |
| ܐܵ | ܒܵܐ | ܒܵ | a |
| ܐܹ | ܒܹܐ | ܒܹ | e |
| ܝܼ | ܒܝܼ | - | i |
| ܘܿ | ܒܘܿ | ܒܿ | o |
| ܘܼ | ܒܘܼ | - | u |
| ܘܸ | ܒܘܸ | - | ū |

此外，还有两个汉语元音
| 元音 | 穆麟德转写 | 拼音 |
| --- | --- | --- |
| ܝ̣ | y | i |
| ܝܘܼ | ioi | ü |

* 元音ܝ̣表示音节zi/ci/si/shi/zhi/chi/ri的元音
* 元音ܝ̣的下标在zi/ci/ri中省略

闭音节末的辅音（即其后没有元音时），可以添加上标ܐ݇表示。

正如叙利亚文字的拼写传统，在正文中元音的上下标记经常省略书写。

## 满语叙利亚字：标准拼写 ܦܘܿܚܿܠܿܢ ܗܹܪܓܹܢ

完整拼写方案有很多冗余的字母。结合满语本身的语音特点，标准拼写对一些元音省略书写，更加适合日常使用满语。

我们观察到满语的以下语音特点：
* 满语的元音a/e/o有很强的元音和谐，它们在词干部分几乎不能混合出现
* 满语的元音i和u是中性元音，可以与其它元音混合出现在同一单词

因此，我们采用以下元音省略规则：
* 元音a/e/o的ܐ和ܘ省略不写，以下情况除外
* 元音词首保留书写
* 单词第一个音节的o保留书写
* 词末n/y/w之后的a/e/o保留书写
* 字母组合ܝܐ和ܘܐ写作ܝܑ和ܘܑ（不在词末时）
* 不在词首时oo拼作ܘ，例如ܒܿܘ和ܩܿܘܠܝ；标注元音时不会与单元音o混淆，因为单元音o的上标标记在ܘܿ上
* 专有名词、汉语转写保留完整拼写

### 辅助符号Syami ܣܝܑܡܝܼ

除直接标注元音外，应用辅助符号syami给单词的元音特征做标注。在省略元音标注的文书中，辅助符号往往更加简明地替代元音标注符号的功能。

上标符号ܐ̈被我们挪用标注带有e/u/ū的音节，用以区分带有a/o的音节。Syami一般标注在e/u/ū之前的辅音上。例如 ܢܝ̈ܥܢܝܑܪܝ和 ܥ̈ܘܢܝܢ分别表示 ܢܝܹܥܢܝܹܪܝܼ和 ܥܘܸܢܝܼܢ。大部分syami可以省略，不影响阅读。

字母ܘ̈表示音节we而非u/ū，词首的u/ū不可以标记syami。

字母组合pe和še使用syami时可写作ܦ̣̈和ܫ̣̈。

例：我 bi ܒܝ，白白 bai ܒܝ̇，碑 bei ܒ̈ܝ，月亮 bia ܒܝܐ，身体 beye ܒܝ̈ܐ，富 bayan ܒܝ̇ܢ ܒܝܑܢ

## 满语叙利亚字：A/W/Y扩展拼写 ܙܘܹܐ ܝܘܿܕ ܗܹܪܓܹܢ

由于作为元音的i/u和作为辅音的y/w在叙利亚系文字里难以区分，我们可以通过添加alef和双写半元音y/w达到区分效果
* 介音i(iy)/u(uw)/ū(ūw)写作i/u/ū
* 复元音中i(iy)/u(uw)/ū(ūw)后的a/e/o保留书写
* 复元音ai/ei和音素组合ay/ey写作yy，ao/eo和aw/ew写作ww，元音a/e标记在其前方的辅音上；由字母ܐ引导时则保留原本拼写

这种拼写相对不依赖元音标记，类似希伯来语新正字，适合阅读辨认。

例：我 bi ܒܝ，白白 bai ܒܝܝ，碑 bei ܒ̈ܝܝ，月亮 bia ܒܝܐ，身体 beye ܒ̈ܝܝܐ，富 bayan ܒܝܝܐܢ

## 样例文本

### 清文启蒙：序（汉语）

清文启蒙一书，乃吾友寿平先生著述，以课家塾者也。其所注释汉语，虽甚浅近，然开蒙循序，由浅入深，行远自迩之寓意焉。况辅迪之初，非此晓畅之文，亦难领会，诚幼学之初筏，入门之捷径也。子尝目睹先生以此课蒙，而稍能颖悟者，学不匝月，即能书诵，且音韵笔画莫不明切端楷，一读不致错误，大有正本清源之义，更见功效捷速之妙。久欲请稿刊刻，以为初学津梁，而先生不许曰，此本庭训小子，设法而作，所注皆系俚言鄙语，粗俗不文，付之梨枣，不无以诒诮乎。子力请再三，始获校梓，其于初学之士，大有裨益云。

雍正庚戌孟春之朔日，作忠堂主人程明远题。

ܚܘܑܠܝܑܣܘܢ ܛܘܒܝ ܙܩܘܫܝ ܐܢܝܐ ܢܝܥܢܝܪܝ ܨܝܢ ܝܢܥܓܝ، ܯܘܿ ܙܘܥ ܛܐܥܢܝ ܒܘܝ ܐܙܢ ܫ̈ܐܥ ܡܝܥ ܝܘܐܢ ܐܪܚ.

### ܫܝܥ ܘ̈ܐܢ ܟܝ ܡ̈ܐܥ ܒܝܬܗ: ܣܝܘܝ ܓܝܣܘܢ (ܙܘܐ ܝܘܕ ܗܪܓܢ)

ܫܝܥ ܘ̈ܐܢ ܟܝ ܡܐܥ ܒܝܬܗ ܣܪܥܓ، ܡܝܢܝ ܓܘܫܘ ܫ̣̈ܐܘ ܦ̇ܝܥ ܣܝܐܢ ܫ̇ܥܢܝ ܦܝܣܡܒܘܡ ܐܪܦܝ، ܒܘܘܝ ܛܫܝܩܘܕ ܛܫܝܒܘܚܥܓ. ܐܕ ܣܘܡ ܣܝܢܕܚ ܢܝܩܢ ܓܝܣܘܢ، ܘܕܘ ܘܡܣܝ ܡܘܐ ܫܝܥܓܝܐ ܒܝܫܝܒ، ܬܢܝ ܛܫܝܪ ܘܪܣܒ ܢܝܝܠܡ ܘܠܗܝܐܢܝ ܝܒ̈ܒܘܪܕ ܡܝܫܝܗܝܐܢܫܝ ܫ̇ܘܡܝܢܕ ܕܘܣܝܢܪ، ܥܘܪ ܝܐܒܘܪܕ ܚܢܫܝܫܝ ܕ̈ܪܝܒܘܪ ܥܘܢܝܢ ܬܒܘܗܒܝ. ܬܪ ܐܥܥܠ ܬܘܟܛܢ ܝܐܪܚܘܕܡ ܛܫܝܒܘܪܕ، ܐܢܬܟ ܓܬܘܟܢ ܝܠ̈ܬܘ ܒܝܬܗ ܘܐܩ ܘܫܝ، ܝܢܘ ܨܪ ܘܠܗܝܪܕ ܡܥܥ ܘܡܒܝ، ܐܪ ܝܐܪܓܝܐܢܝ ܐܙܝܥܢ ܛܫܝܪ ܘܪܣܝܝ ܣܘܫܘܥܥ ܕܘܘܒܘܪ ܐܕ، ܕ̈ܘܩ ܕܘܣܝܡܒܘܪ ܕܘܩ ܙܘܥܘܢ ܩܝܝ. ܒܝ ܟܡܘܢܝ ܣܝܐܢ ܫ̇ܥܢܝ ܐܪܒ ܙܦܦܝ ܐܙܝܥܢ ܘܪܣܒ ܛܫܝܒܘܚܒ ܬܘܐܫܝ ܡܙܝܓ ܣܘܪܟܢ ܘܠܗܝܫܘܥܥ ܘܪܣ ܘܫܝ، ܛܫܝܡ ܒܝܐܠܡ ܒܝܝܒܘܪܩܘ، ܘܬܚܝܝ ܚܘܠܡ ܐܪܡ ܡܘܬܡܒܝ ܣܪ ܐܥܥܠ، ܙܝܠܥܢ ܡܘܕܢ ܓܬܘܟܢ ܒܘܠܥ، ܗܪܓܢܝ ܙܝܙܘܢ ܛܘܒ ܛܘܢܕ ܘܙܪܩܘܥܓ ܐܩܘ، ܝܐܠ ܐܡܓܪܝ ܛܫܝܦܝ ܫܠܒܘܪ ܙ̈ܫܗܘܢܙܪܕ ܝܣܝܢܪܩܘ ܘܡܒܝ، ܦܘܠܗܒ ܛܘܒ، ܣܟܝܐܢܒ ܒܘܠܥ ܘܒܘܪ ܥܘܢܝܢ ܐܡܒܘܠ ܒܝܣܝܪ ܕܕ، ܓܡܘ ܓܘܥ ܬܘܨܒ ܚܘܕܘܢ ܘܙܪ ܦ̈ܪܓܘܐܫܘܟ ܒܒܝ. ܒܝ ܐܝܦܝܢܝ ܙܝܣ̈ܒ ܥܝܝܦܝ ܦܘܠܒܘܦܝ، ܬܢܝܟܢ ܛܫܝܪ ܘܪܣܕ ܕܘܪܘܢ ܬܘܐܩܘ ܘܒܘܟܝ ܣܡ ܒܝܝܚܕ، ܣܝܐܢ ܫ̇ܥ ܘܚܩܘ ܗܢܕܘܗܥܓ، ܐܪ ܒܝܬܗ ܣܪܥܓ ܡܝܢܝ ܒܘܘܝ ܙܘܣܒ ܛܫܝܗܝܐܡ ܦܩܣܝܩܢܝ ܥܡܡ ܐܪܚܥܓ، ܣܘܗܥܓ ܝܘܘܢܝ ܒܘܝܐ ܚܠܥܪܝ ܓܝܣܘܢ ܡܘܐ ܐܠܒܬܘ ܒܝܡ ܦܝܐܢ ܐܩܘܥܓ، ܦܘܠܒܘܫܝ ܒܣܘܫܘܢ ܐܩܘ ܣܡܘܘ ܣܗ ܒܝܗ. ܒܝ ܕܚܘܢ ܕܚܘܢܝ ܚܘܣܘܬܘܠܡ ܒܝܝܪ ܙܚܕ، ܬܢܝ ܒܚܦܝ ܐܫܒܘܡ ܦܘܠܒܘܚ، ܐܕ ܛܙܝܥܢ ܘܪܣ ܛܫܝܪܕ، ܐܝܢܫܝ ܐܡܒܘܠ ܢܝܐܫܫܘܢ ܬܘܨ ܘܙܪ ܒ ܒܝܕܪ.

ܚܘܐܠܝܐܣܘܢ ܛܘܒܝ ܙܩܘܫܝ ܐܢܝܐ ܢܝܐܥܢܝܐܪܝ ܨܝܝܢ ܝܢܥܓܝ، ܯܘܿ ܙܘܥ ܛܐܥܢܝ ܒܘܘܝ ܐܙܢ ܫ̈ܐܥ ܡܝܥ ܝܘܐܢ ܐܪܚ.
